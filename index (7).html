Retiro anual — bruto necesario para cada gasto neto objetivo

Para cada año muestra cuánto hay que retirar bruto para que te quede el gasto neto objetivo inflado. El % ganancia sobre cada retiro evoluciona año a año: el coste base no crece pero el valor sí, así que cada año tributa una fracción mayor.

Año Edad Neto obj./mes Neto obj./año % ganancia Bruto necesario Impuesto Tipo ef. Cartera proj.
% ganancia automático Cada año se recalcula: coste base fijo (FIFO real) dividido por valor proyectado creciente. A más años de retiro, mayor fracción tributa.
Bruto necesario Iteración numérica: encuentra el retiro bruto que, tras aplicar IRPF sobre la parte de ganancia, deja exactamente el neto inflado objetivo.
Etapas de gasto El neto objetivo cambia según las etapas definidas en Supuestos — retiro activo, vida estable, salud.
Orientativo No sustituye asesoramiento fiscal real. Tramos IRPF 2026.
